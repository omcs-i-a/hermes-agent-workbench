# Experiments

This directory stores repeatable AI Agent research experiments.

Recommended experiment directory name:

```text
YYYY-MM-DD-short-topic/
```

Recommended contents:

```text
prompt.md          # task prompt used for the experiment
setup.md           # environment, tools, model, and assumptions
run-log.md         # execution notes and observations
result.md          # final output or artifact summary
review.md          # evaluation using a rubric from evals/rubrics/
artifacts/         # optional raw outputs, screenshots, traces, data
```

Experiments should be reproducible enough that another agent can rerun or critique them.
